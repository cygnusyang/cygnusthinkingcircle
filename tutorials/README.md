## 💡 核心理念

**写一次，处处发布**

- **知识库 (knowledge-base/)** 作为唯一真相源
- **博客 (cygnusyang.github.io/)** 作为主要展示平台
- **多平台适配** 通过 LLM 自动实现

## 🚀 核心特性

- ✅ 基于 Markdown 的轻量级知识管理
- ✅ Hugo + FixIt 主题的高性能博客
- ✅ LLM 驱动的跨平台内容适配
- ✅ Claude Code 智能开发助手集成
- ✅ Git 版本控制，随时随地备份
- ✅ 免费的 GitHub Pages 托管
