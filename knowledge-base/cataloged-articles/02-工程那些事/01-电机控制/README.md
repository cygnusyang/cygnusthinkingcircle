# 无刷电机控制学习路径

## 📚 学习目标

系统掌握无刷电机（BLDC）控制原理，从基础理论到 FOC 实现到工程实践。

---

## 🗺️ 学习阶段

### 第一阶段：基础理论（2-3周）

#### 1.1 电机原理
- [ ] 无刷电机结构和工作原理
- [ ] 与有刷电机的区别
- [ ] 反电动势波形（梯形波 vs 正弦波）
- [ ] 转矩生成原理：`τ = K_t × I`
- [ ] 最大力条件：θ = 90°

#### 1.2 坐标变换
- [ ] 三相静止坐标系 (abc)
- [ ] 两相静止坐标系 (αβ) - Clarke 变换
- [ ] 两相旋转坐标系 (dq) - Park 变换
- [ ] 逆变换实现
- [ ] 为什么需要坐标变换

#### 1.3 调制技术
- [ ] SPWM（正弦脉宽调制）
- [ ] SVPWM（空间矢量脉宽调制）
- [ ] 六扇区划分
- [ ] 矢量合成原理

### 第二阶段：控制算法（3-4周）

#### 2.0 控制算法总览（必读）
- [ ] 控制算法分类与对比
- [ ] 方波控制 vs SPWM vs FOC vs DTC vs SMC vs MPC
- [ ] 传统算法 vs 现代算法 vs 智能算法
- [ ] 算法选择决策流程
- [ ] **前沿研究方向**：AI/深度学习、自适应控制、无传感器

#### 2.1 FOC（磁场定向控制）
- [ ] FOC 核心原理
- [ ] d-q 轴解耦控制
- [ ] 最大转矩/安比控制 (`i_d = 0`)
- [ ] 电流采样与重构
- [ ] 转子位置估计

#### 2.2 三环控制
- [ ] **电流环**（内环，最快）
  - 采样频率：20kHz+
  - PI 控制器设计
  - 带宽和相位裕度
- [ ] **速度环**（中环）
  - 采样频率：1-5kHz
  - 速度估计与滤波
- [ ] **位置环**（外环，最慢）
  - 采样频率：100-1kHz

#### 2.3 PID 整定方法
- [ ] Ziegler-Nichols 方法
- [ ] 极点配置法（电流环）
- [ ] 手动整定经验
- [ ] 串级控制整定顺序

## 🔑 关键概念速查

```
FOC 最大转矩条件: i_d = 0, i_q = τ/K_t

Clarke 变换:
  i_α = i_a
  i_β = (1/√3)(i_b - i_c)

Park 变换:
  i_d = i_α cos(θ) + i_β sin(θ)
  i_q = -i_α sin(θ) + i_β cos(θ)

三环带宽比例:
  电流环 > 速度环 > 位置环
  约 10:1 带宽差
```
